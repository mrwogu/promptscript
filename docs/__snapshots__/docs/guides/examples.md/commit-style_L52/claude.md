# CLAUDE.md

## Examples

### Example: feat-commit

Feature commit with scope

**Input:**

```
Added user authentication with JWT tokens
```

**Output:**

```
feat(auth): add JWT-based user authentication
```
### Example: fix-commit

Bug fix commit

**Input:**

```
Fixed null pointer in payment service
```

**Output:**

```
fix(payments): resolve null pointer in charge handler
```
